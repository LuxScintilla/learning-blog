My name is Ekaterina Furman, welcome to my Learning Blog!
Here, I have dedicated a space to keep a weekly diary of my learning progress with The Odin Project. I have done my best to keep track of the extra resources I used along the way, which are listed along with the subjects learned on a given week.
The only experience I had, prior to my start with The Odin Project, was some basic HTML and CSS.
